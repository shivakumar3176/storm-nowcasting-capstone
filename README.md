#  Storm Nowcasting using Deep Learning (ConvLSTM)

##  Overview

This project focuses on predicting severe convective storms using deep learning techniques. The model analyzes spatio-temporal radar data to forecast future storm evolution.

##  Key Features

* ConvLSTM-based spatio-temporal modeling
* Encoder-decoder architecture
* Multi-step storm prediction
* Evaluation using SSIM, CSI, POD, FAR
* Severity-based storm analysis

##  Model Architecture

* ConvLSTM layers for temporal learning
* CNN-based spatial feature extraction
* Autoregressive prediction of future frames

##  Dataset

* SEVIR Dataset (Storm Event Imagery)
* Radar-based input sequences

##  Tech Stack

* Python
* PyTorch
* Google Colab

##  Evaluation Metrics

* MSE (Mean Squared Error)
* SSIM (Structural Similarity Index)
* CSI (Critical Success Index)
* POD (Probability of Detection)
* FAR (False Alarm Ratio)
